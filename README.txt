SSUK-Database

This is the readme for the SSUK-Database, started 24/08/22 (However coding was done before this date).

-Created by Connal Crighton


The main goal of this program is to be able to open and edit a MySQL database, multiple users must be able to open
different databases and be able to change and save data.


Before starting the program install these library's
py -m pip install mysql-connector-python
py -m pip install sqlalchemy
py -m pip install tk
py -m pip install mysqlclient
py -m pip install PyMySQL
